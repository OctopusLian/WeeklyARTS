## 父亲分羊思路解析  
```
令n’ = n + 1，我们可以认为聪明的邻居“分”到了1/n'的遗产。实际上我们在求不定方程 
   1/x + 1/y + 1/z + 1/n’ = 1
的整数解。为了简化问题，我们设x < y < z（思考：如果允许相等会怎样？） 有朋友这时就抓起键盘来编码了，还有的朋友想到了埃及分数。其实我们这时还可以再深入思考一下。首先x不会小于2，否则老大分得了全部的遗产；那么x的上限是多少呢？由于：
1/x + 1/x + 1/x + 1/x  >  1/x + 1/y + 1/z + 1/n’ = 1
所以4/x > 1，因此x < 4。这样我们就得到了x的范围：[2, 3]
 
给定x后，y的下限就是x+1，由于：
1/y + 1/y + 1/y  >  1/y + 1/z + 1/n’ = (1 - 1/x)
所以3/y > (1 - 1/x)，因此y < 3 / (1 - 1/x)。这样我们就得到了y的范围：[(x + 1) .. 6]
 
用同样的方法，我们可以推出z的范围是：[(y + 1) .. 12]
 
现在我们可以转向编程的方法了。我们可以在上述范围内穷举x、y、z:
 
    [(x, y, z, n) | x in [2, 3], y in [(x+1) .. 6], z in [(y+1) .. 12], n = 1 / (1 – (1/x + 1/y + 1/z)) - 1, n > 0]
 
这个程序只有一行，如下面的Haskell代码：
 
[(x, y, z, n) |
       x <- [2, 3],
       y <- [(x + 1) .. 6],
       z <- [(y + 1) .. 12], let d = x * y * z - y * z - x * z - x * y, d > 0,
       let n = (y * z + x * z + x * y) `div` d,
       (n + 1) `div` x + (n + 1) `div` y + (n + 1) `div` z == n]
 
运行这一程序，我们可以得到全部7组解:
 
[(2,3,7,41), (2,3,8,23), (2,3,9,17), (2,3,12,11), (2,4,5,19), (2,4,6,11), (2,4,8,7)]

张景中院士在他的文章中，给出的方法于此类似，只是最后仍然依赖手工计算，而不用求助于计算机。
```
