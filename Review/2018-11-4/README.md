#### 我知道是什么给我带来了成功而且我想和你分享  
你应该选择什么语言？  
1,C++――强烈推荐！它非常快。因为STL，用C++实现不同的算法所需的时间都很少。所有比赛都接受C++。从我写下第一行代码，我一直使用它。  
2,C――去学习C++，因为STL。如果你了解C，那么你也可以准备使用C++编码。  
3,JAVA――它很慢。但是它有Big Integer类，尽管很少有问题需要用到它。如果时间紧迫，你将会超出时间的限制。  
 
你可以在哪里练习？  
我推荐Sphere Online Judge（SPOJ）。它在题目质量和数量上都很有效，如果你在解决问题时遇到困难，可以在线获取解决方案和社区讨论。
