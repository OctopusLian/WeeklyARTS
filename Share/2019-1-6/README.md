完善流媒体服务功能，使其可以顺利播放支持TCP协议的视频流。  

### 二、所需技能概述  
如果要完成该项任务，所需掌握的基础有以下几项：

- 计算机网络基础，尤其是HTTP协议，和TCP/IP协议；  
- hls协议（HTTP Live Streaming）。
